Anotações da minha evolução no curso Git/Github

1º Passo:
- Estou fazendo o BootCamp da TQI;
- Cheguei no Módulo Introdução ao Git e Github;
- Completei o módulo e ficou restando fazer o projeto.

2º Passo:
- Fiz a instalação do Git através do link  http://git-scm.com/download/win ;
- Criei minha conta no GitHub https://github.com/join/plan?plan=free.

3º Passo:
- Utilizei meus conhecimentos adquiridos no curso de introdução ao Git/Github da DIO para criar meu Primeiro Repositório.

4º Passo: 
- Estudei a documentação do Git https://git-scm.com/book/pt-br/v2 e aprendi mais alguns comandos que não foram abordados no curso dentre eles o Branch.

5º Passo:
- Aprendi a fazer Branch e criei um novo;

6º Passo:
- Aprendi a fazer Merge:
  1- git pull da branch principal
  2- criar uma nova branch
  3- fazer as atualizações na nova branch
  4- finalzar o trabalho na nova branch
  5- git checkout na principal
  6- git pull
  7- Merge da branch temporária com a branch principal
  8- git push da banch principal

7º Passo:
- Aprendi a fazer uma Pull request

8º Passo:
- Aprendi o Git Ignore para "esconder" arquivos e pastas
  - touch .gitignore
